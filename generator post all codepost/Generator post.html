FAQ: README

1. Apa itu Format Postingan?

Format postingan adalah Code Post yang dapat kamu buat untuk menampilkan data anime. kamu dapat menggunakan {{data}} (yang diambil dari API.) untuk mengisi data code post kamu.

2. data Apa yang Tersedia?

  • {{title}}: Judul anime.
  • {{description}}: Deskripsi anime.
  • {{bannerImage}}: URL gambar banner anime.
  • {{coverImage}}: URL gambar cover anime.
  • {{studios}}: Daftar studio yang memproduksi anime.
  • {{genres}}: Daftar genre anime.
  • {{episodes}}: Jumlah episode anime.
  • {{duration}}: Durasi per episode anime.
  • {{popularity}}: Popularitas anime (nilai numerik).
  • {{averageScore}}: Rata-rata skor anime (nilai numerik).
  • {{characters}}: Daftar karakter anime, termasuk gambar dan nama mereka.
Lebih lengkap silahkan cek contohnya di FAQ 'Contoh Format Postingan'

MAL

  • {{title}}: Judul utama anime.
  • {{coverImage}}: URL gambar cover dari anime.

  • {{type}}: Jenis atau format anime (misalnya: TV, Movie, OVA).
  • {{episodes}}: Jumlah total episode anime.
  • {{duration}}: Durasi rata-rata setiap episode dalam menit.
  • {{status}}: Status anime (misalnya: Finished, Ongoing).
  • {{aired}}: Rentang tanggal penayangan anime.
  • {{premiered}}: Musim dan tahun pertama kali anime ditayangkan (misalnya: Spring 2023).
  • {{broadcast}}: Informasi tentang waktu siaran anime (misalnya: setiap hari Rabu pukul 22:00).
  • {{averageScore}}: Skor rata-rata dari penilaian pengguna.
  • {{popularity}}: Popularitas anime berdasarkan jumlah penonton atau penggemar.
  • {{studios}}: Studio animasi yang memproduksi anime.
  • {{producers}}: Perusahaan produser yang terlibat dalam produksi anime.
  • {{licensors}}: Perusahaan yang mendapatkan lisensi distribusi anime di luar negeri.
  • {{genres}}: Genre atau kategori utama anime (misalnya: Action, Comedy, Drama).
  • {{demographic}}: Target penonton berdasarkan demografi (misalnya: Shounen, Seinen).
  • {{source}}: Sumber data atau cerita asli anime (misalnya: Manga, Novel, Original).
  • {{sourceID}}: ID sumber informasi anime (misalnya: ID pada database seperti MyAnimeList atau Anilist).
  • {{rating}}: Klasifikasi usia atau rating anime (misalnya: PG-13, R-17).

4. Bagaimana Cara Menggunakan Generator Post ini?

  1. Langkah 1: Tulis format postingan di textarea yang disediakan.
  2. Langkah 2: Gunakan {{data}} di dalam format post sesuai dengan data yang ingin ditampilkan.
  3. Langkah 3: Klik tombol "Save Format post" untuk menyimpan format ke localStorage.
  4. Langkah 4: Pilih Sumber ID Anime yang akan kamu gunakan
  5. Langkah 5: Masukan ID Anime kedalam input "ENTER ID" dan Click "GetPost"
  6. Langkah 6: Silahkan double cek data yg didapat apakah sudah benar, jika sudah maka copy dan paste ke blog kamu.(selesai)

5. Bisakah Mengubah Format Postingan Setelah Menyimpannya?

Ya, Kamu dapat mengubah format postingan kapan saja. Cukup masukkan format baru di textarea dan simpan lagi dengan judul yang sama. Format baru akan menggantikan yang lama.

6. Bagaimana Jika Aku Ingin Menghapus Format yang Disimpan?

Untuk menghapus format yang disimpan, Kamu dapat memilih format mana akan dihapus terlebih dahulu, setelah memilih maka click tombol hapus yg tersedia

7. Bisakah membackup Format Postingan?

Ya, Kamu dapat mencadangkan format postingan kapan saja. cukup click tombol "Backup" dan tunggu beberapa Detik..file akan siap untuk didownload.

Bagaimana Memulihkan data backup?

Sangat Sederhana.. pertama click "pilih file" lalu cari dan pilih file hasil backup sebelumnya.. lalu click "pulihkan" hasil data backup akan kembali pulih
