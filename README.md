     <li><strong>{{chapters}}:</strong> Chapter (Manga)</li>
     <li><strong>{{volumes}}:</strong> volumes (Manga)</li>

    <li><strong>{{characters}}:</strong> Daftar characters dan voice actors (VA) Jepang.</li>
